# SAP - Roadmap

This roadmap outlines the planned evolution of SAP (Sia's Admin Panel).
It reflects intended development direction and may change as the project evolves.

--- 

# 🎯 Project Vision

SAP is evolving from an advanced admin panel into a structured moderation framework designed for secure, scalable Roblox experiences.

The focus moving forward is:
- Security hardening
- Moderation infrastructure depth
- Cross-server control
- Discord integration
- Professional-grade tooling

---

# 🚧 Current Phase (Post v0.6.0)

SAP core moderation systems are complete.

The next development focus is infrastructure expansion and ecosystem integration.

---

# 🔜 Upcoming Milestones

## Discord Integration
- Webhook-based moderation logs
- Structured log formatting
- Action-specific log embeds
- Optional bot integration groundwork

---

## Role & Permission System
- Hierarchical role structure
- Granular permission control
- UI restriction per role
- Secure server-side enforcement

---

## Warning & Escalation System
- Persistent warning tracking
- Automatic escalation logic
- Integration with moderation history

---

## Moderation History & Case System
- Full player moderation history viewer
- Case ID system
- Evidence tracking support
- Filter & search functionality

---

## Cross-Server Moderation Framework
- Global mute system
- Cross-server action synchronization
- Centralized moderation control layer
- Global announcement system

---

# 🔮 Long-Term Direction

- Moderation analytics dashboard
- Advanced session intelligence panel
- Admin action confirmation safeguards
- UI customization & theming
- Stability and performance optimizations
